# GreenOps 101 — Digital Sustainability for IT Practitioners and Leaders

A practitioner accreditation course developed by the **TBM Council**. Fourteen modules across three tracks, designed for technology professionals who need to understand digital sustainability as an operational discipline rather than a reporting exercise.

## Course structure

| Track | Modules | Audience |
|-------|---------|----------|
| Track 1 — Technical Foundation | M1–M4 | All practitioners |
| Track 2 — Operational Practice | M5–M8 | Engineers, procurement, product |
| Track 3 — Leadership and Strategy | M9–M14 | CIOs, CTOs, programme leads |

## Running locally

Download all files to a single folder. Open `index.html` in a browser. All navigation is relative — no server required.

## Live site

Hosted via GitHub Pages at: `https://[org].github.io/greenops-101/`

## Interactive elements

- Module 0 (index.html): Course hub with interactive track navigator and progress tracking
- Module 1: Waste category explorer, Scope diagram, Action model accordion
- Module 2: Dimension explorer, lifecycle phase diagram
- Module 3: Carbon intensity bar chart + **live NESO API regional data**
- Module 4: 7-domain selector, PUE/utilisation scatter chart
- Module 5: Rebound effect animation, **live NESO scheduling forecast**, trade-off triangle
- Module 6: SCI live calculator
- Module 8: Multi-dimensional maturity radar self-assessment
- Module 10: Year-1 business case builder (scalable)
- Module 11: 90-day Horizon 1 sprint tracker (progress stored in browser)
- Module 12: 5Cs leadership self-assessment
- Module 13: Scope 4 avoided emissions calculator

All modules include knowledge check questions and Pause & Reflect prompts drawn from the course slide content.

## Live data

Modules 3 and 5 call the [National Grid ESO Carbon Intensity API](https://carbonintensity.org.uk) directly from the browser. No server required. Falls back gracefully when the API is unavailable.

## Accreditation

The GreenOps Practitioner Foundation accreditation is available on completion of all 14 modules and passing the accreditation test. Details at [tbmcouncil.org](https://tbmcouncil.org).
